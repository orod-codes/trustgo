# TRUCKNGO App

Vite + React 18 + TypeScript + Tailwind 4 + React Router 7.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Resident app routes

| Route | Screen |
|-------|--------|
| `/resident/login` | Onboarding splash |
| `/resident/signin` | Sign in |
| `/resident/home` | Dashboard |
| `/resident/track` | Live truck map |
| `/resident/sort` | Upload waste + camera |
| `/resident/wallet` | Points wallet |
| `/resident/marketplace` | Rewards shop |
| `/resident/profile` | Profile |

## Portals

- **Driver** — `/driver/login` → route execution
- **Admin** — `/admin/login` → fleet & analytics

## Assets

Place branding images in `public/`:

- `logo.png` — app icon
- `truck.png` — hero truck illustration
- `city.png` — city/onboarding art
