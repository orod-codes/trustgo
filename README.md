# TRUCKNGO

Smart waste collection app for residents, drivers, and municipal admins.

The full application lives in **`TRUCKNGO/`** (the old `Pharma/` folder was merged here).

## Quick start

```bash
npm run install:app
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Demo logins

| Portal | Credentials |
|--------|-------------|
| Resident | `R100001`, `R100002`, or `R100003` on sign-in |
| Driver | `driver@truckngo.et` / `driver123` |
| Admin | `admin@truckngo.et` / `admin123` |

TRUCKNGO/
